# Bio-Economic Feed Ration Optimizer
A high-performance decision support application for precision cattle finishing that replaces static nutritional calculators with dynamic, market-aware optimization. The application combines the HiGHS linear programming solver to mathematically minimize diet costs across 20+ non-linear NASEM 2016 biological constraints within milliseconds, while integrating a continuous feed of CME Live Cattle futures to construct real-time marginal revenue S-curves. By computing declining feed efficiencies against future price projections, it algorithmically identifies the exact Day 1 to Day N profit peak intersection (Marginal Cost = Marginal Revenue), allowing producers to manipulate ADG via the interactive dashboard and optimize their exact slaughter date for maximum return on invested capital.
